Featured Project

This was a small side project to work on some automation using NASA's Astronomy Picture of the Day (APOD) API. Check it out here.

It sends messages through a reconstructed SMS/MMS service built by Alfredo Sequeida.

The APOD Bot sends a daily message based on a preset timezone, mobile provider, and phone number. It pulls data from the NASA API, reformats it, and sends it through my MMS system.

My next step for this project is to create a site where individuals can register for the texting service.
